Lors de l'achat d'un Theme Elana, vous avez plusieurs type de licence en fonction de votre contrat. Vous trouverez le détail de votre licence en vous connectant à votre compte.
>>>>> Il existe des licences standard et développeur pour l'achat d'un thème "Stand-Alone", c'est à dire sans prestations d'installation sur serveur ou personnalisation. Si vous avez opté pour une prestation tout en un (personnalisation, installation, maintenance) la licence peut différer.

<h2>Questions fréquentes</h2>

<div class="uk-accordion" data-uk-accordion>

    <h3 class="uk-accordion-title">1. Ai-je besoin d'une licence supplémentaire si je veux utiliser un thème Elana sur un second site ?</h3>
    <div class="uk-accordion-content">
    <p>Oui si vous avez une licence standard, il vous faudra acquérir une seconde licence, dans le cas de licences multi-site ou développeur vous pouvez installez le thème Elana respectivement sur 3 sites et un nombre illimités.</p>
    </div>

    <h3 class="uk-accordion-title">2. Puis-je utiliser un thème Elana sur un site de test, puis le déplacer vers un site Live lorsque qu'il est terminé ?</h3>
    <div class="uk-accordion-content">
    <p>Oui, vous pouvez utiliser un test de site et le déplacer une fois terminé.</p>
    </div>

    <h3 class="uk-accordion-title">3. Puis-je garder un site de test et un site live avec une seule licence ?</h3>
    <div class="uk-accordion-content">
    <p>Non, si votre site de test est en live, et oui s'il est en local</p>
    </div>

</div>
